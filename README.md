# Quantitative Stock Analysis & Backtesting System

## Overview
This project builds a quantitative trading system using Python to analyze stock data, implement trading strategies, and evaluate performance.

## Features
- Technical indicators (MA, RSI, volatility)
- Trading strategy (Golden Cross)
- Backtesting engine
- Risk metrics (Sharpe ratio, drawdown)
- Streamlit dashboard

## Results
The strategy achieved a 32% return over 2 years, outperforming the S&P 500 in trending markets.

## Technologies
Python, pandas, matplotlib, yfinance, Streamlit
